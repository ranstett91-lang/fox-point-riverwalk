Why this matters for Amtrak

The water is rising toward the tracks.

Wilmington is sinking faster than any other shoreline area on the Northeast Corridor. The Delaware River is also rising. Fox Point sits right where those pressures meet the rail line.

Project Readiness

This project already has a clear scope and next steps.

This is not just a big idea. The state has already named the site, outlined the work, and shown the next steps.

State Status

DNREC already manages Fox Point Park Phase II as an active site.

Delaware's environmental agency, DNREC, already lists this as a managed cleanup area. The project area covers about 203 river acres across 1.4 miles, including Fox Point Cove.

Planned Work

Fox Point Cove is already named as the restoration project.

The state plan calls for marsh restoration at Fox Point Cove, including invasive plant removal, shoreline grading, clean fill, native planting, wave-break structures, and erosion control near Stoney Creek Wharf.

Next Steps

The next step is design, permits, and construction planning.

The plan already covers about 2.7 acres of wetland work. Next comes engineering, permits, construction, and three years of follow-up monitoring.

Funding

About $5.03M is already tied to this site. The missing piece is the work to use it.

This site does not need a new pitch. It needs design, permits, planning, and people working together so the money already linked to Fox Point can be put to work.

$2.5M

Heffernan Bond Bill

Not active yet

Wetland and park work

Rep. Heffernan pledged $3M in bond funding to help meet state environmental requirements for the new Edgemoor port. Of that, $2.5M is set aside for wetland and park work at Fox Point. It would move when port construction starts and the Diamond State Port Corporation approves it.

$1.725M

DNREC restoration funds

In state plan

Fox Point Cove wetland restoration

The January 9, 2025 state plan names Fox Point Cove as the project site. It covers wetland work, erosion control near the wharf, design, and permits. If money is left over, it could support more restoration or later upkeep.

$808K

Chemours / NOAA settlement

Not assigned yet

Future restoration money from a pollution settlement

This money comes from a settlement tied to pollution at the former Chemours titanium dioxide plant in Edgemoor. NOAA leads the process. No site has been chosen yet, but Fox Point fits the kind of work this money could support.
