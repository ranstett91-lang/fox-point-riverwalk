Project Readiness

This is already a scoped project with a delivery path.

This is not a blank-slate idea. DNREC already has a final restoration plan, the geography is defined, and the next need is the project work that moves the plan toward permits and delivery.

Regulatory Status

Fox Point Park Phase II is already an active DNREC site.

Fox Point Park Phase II is in long-term stewardship. OU-2 covers about 203 acres of Delaware River shoreline, including Fox Point Cove.

Defined Scope

Fox Point Cove is already named in the restoration plan.

The final restoration plan calls for marsh restoration, native planting, living shoreline features, and targeted erosion control at Stoney Creek Wharf.

Delivery Path

The next step is design, permits, and execution.

The conceptual plan covers about 2.3 acres of restoration and 0.4 acre of enhancement. The work now is engineering, permitting, construction planning, and monitoring.

Capital Already Identified

$5.03M is already in the pipeline.

These dollars do not need a new vision. They need design, permitting, sequencing, and partner coordination so the money can actually move.

$2.5M

Heffernan Bond Bill

Contingent

Wetland creation + park improvements

Rep. Heffernan pledged $3M in bond funds for state mitigation tied to the new Edgemoor port. Of that, $2.5M is aimed at Fox Point wetland and park improvements, pending port construction and approval by Diamond State Port Corporation.

$1.725M

APU / DNREC plan

Final Plan

Fox Point Cove tidal marsh restoration and enhancement

The final restoration plan, dated January 9, 2025, names Fox Point Cove as the restoration site. The plan covers marsh restoration, shoreline work, design, permitting, and related improvements.

$808K

Chemours / NOAA

Future Process

Environmental restoration — Titanium Dioxide legacy

A federal and state settlement tied to the Chemours titanium dioxide site set aside $808K for restoration. No project has been chosen yet, but Fox Point fits the purpose well.
