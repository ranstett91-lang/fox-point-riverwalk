# Security Policy

## Supported Version

The `main` branch is the supported version of this project.

## Reporting a Vulnerability

Please do not open a public issue for suspected security vulnerabilities.

For sensitive reports, contact the repository owner privately through GitHub first and include:

- a short description of the issue
- steps to reproduce it
- any affected URL or file
- the potential impact

If the issue is confirmed, we should fix it promptly and disclose only the minimum needed after a patch is available.
